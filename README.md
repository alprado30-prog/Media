# 📊 Calculadora de Média C#

Projeto simples em C# para calcular a média de 3 notas e informar se o aluno foi Aprovado, Recuperação ou Reprovado.

## 🚀 Como funciona
O programa pede 3 notas para o usuário, calcula a média e mostra o resultado:

- **Aprovado**: Média >= 7.0
- **Recuperação**: Média >= 5.0 e < 7.0  
- **Reprovado**: Média < 5.0

## 🛠️ Tecnologias
- C#
- .NET Console Application
